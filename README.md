# 3-Month Personal Budget Analysis (Excel Dashboard)

## Overview
This project analyzes 3 months of personal spending data to identify trends, compare monthly changes, and highlight key insights.

The goal was to transform raw data into a clean and simple dashboard that shows spending behavior clearly.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Basic Excel functions (SUM, percentages)
- Data visualization (line chart)

---

## Dataset
Includes spending data for:
- November
- December
- January

---

## Key Insights
- December had the highest spending at $3,003 (22% increase from November)
- Spending decreased by 19% in January
- Rent remained the largest consistent expense
- Miscellaneous spending increased in December
